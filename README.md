# PPDS ERP

Pondok Pesantren Darussalam Sumedang Enterprise Resource Planning (ERP).

Sistem terintegrasi untuk seluruh aktivitas pondok pesantren dalam satu platform tunggal.

## Screenshot

![Dashboard Placeholder](https://via.placeholder.com/800x400?text=Dashboard+PPDS+ERP)

## Tech Stack

- Framework: Next.js (App Router)
- Language: TypeScript
- Database: Cloudflare D1
- ORM: Drizzle ORM
- UI: Tailwind CSS + Shadcn UI

## Requirement

- Node.js 20+
- npm 10+
- Wrangler CLI (for Cloudflare D1)

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
